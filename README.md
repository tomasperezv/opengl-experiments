# opengl-experiments
Repository containing experiments with OpenGL 3D graphics

# Installation

```bash
$ apt-get install g++ freeglut3-dev glew1.5-dev libmagick++-dev libassimp-dev libglfw-dev
$ ln /usr/lib/pkgconfig/libglfw.pc /usr/lib/pkgconfig/glfw3.pc
```
